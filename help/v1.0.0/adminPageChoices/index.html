Admin Page Choices (Help)

Note: These features are only available to select people who oversee the OED site (called admins) so these features are not usually of interest to a general user.

Overview

In addition to choosing the type of graphic to see, OED maintains several other pages for admins to maintain OED.

Usage

There are several pages available to admin users that are accessed by the "Pages" dropdown menu in the top, right corner of the web browser window (highlighted with a blue box) as shown in the followings figure:

The dropdown choices are given next with a brief description where the provided link goes to another help page with extensive information on that page.

  • Conversions. This shows details on each conversion along with the ability to edit or create a new one.
  • CSV. This allows for upload of a CSV file with meters or readings.
  • Groups. This shows details on each group along with the ability to edit or create a new one.
  • Maps. This shows details on each map along with the ability to edit or create a new one.
  • Meters. This shows details on each meter along with the ability to edit or create a new one.
  • Units. This shows details on each unit along with the ability to edit or create a new one.
  • Admin Panel. This shows details on site preferences and users along with the ability to modify them.
  • Help. Opens this help page.
