Hide Options (Help)

Overview

There are times you want to have more space for graphics by docking the options/menus normally listed on the left-hand side of the OED pages. This may be desired when you have a smaller screen/resolution, you are displaying on a monitor without interactivity or you simply want more space for the graphics. When you dock these items, they can be undocked from the "Menu" button on the top, right corner of the OED screen where there is an option to "Show options" (undock) the options/menus.

Usage

OED will automatically dock the menus if the size of your web browser window is too small. This can happen due to resizing the window or from the screen resolution/size. When this is done there will not be the option to either hide or show options.
