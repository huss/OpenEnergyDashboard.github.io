The following assumes as you have an understanding of general units.

OED supplied units

While a site can create any desired unit and conversion, OED wants to make setup as easy as possible. To facilitate this, OED provides certain units and conversions as part of its installation. The following units and conversions are provided by OED.

Units included

Unit Type Represents Displayable
kWh unit quantity yes
Electric_Utility meter quantity no
BTU unit quantity no
m³ gas unit quantity no
kg unit quantity no
metric ton unit quantity no
gallon unit quantity no
liter unit quantity no
Fahrenheit unit raw no
Celsius unit raw no

Without action by the site admin, the only graphing unit visible to a user is kWh. A meter unit Electric_Utility is usable kWh meters because those are never directly displayable to the user. The other units can be made available as a graphing unit if the displayable is changed to all.

Conversions included

Source unit Destination unit Bidirectional Slope Intercept
kWh BTU yes 3412.142 0
BTU m³ gas yes 2.73e-5 0
kg metric ton yes 0.001 0
liter gallon yes 0.2641729 0
Celsius Fahrenheit yes 1.8 32
Electric_Utility kWh no 1 0

Most of these conversions will not be visible to the user until the units involved are made visible to the user. The conversion of Electric_Utility to kWh is relating the Electric_Utility meter unit to kWh in which it reads so the slope is 1 as no conversion is effectively applied. The values for slope and intercept should be fairly standard ones that all sites are comfortable using but can be edited if desired. Note that chained conversions means that all the energy units (kWh, BTU, m³ gas) can be used and converted between.
