OED Language selection (Help)

Overview

Each OED site has a default language that is used to display information. The user can select from other available languages to have OED display information in an alternative language.

Usage

The "Select a language type" dropdown menu on the home screen is shown in the next figure (highlighted in a blue box).

At the current time the user can select English, French and Spanish.

Details

OED is interested in including more languages within the OED system. If you are interested in helping translate OED to another language then please contact us.
